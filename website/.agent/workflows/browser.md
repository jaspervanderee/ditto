// turbo-all

The user uses **Brave Browser** instead of Chrome. When using the browser subagent or `open_browser_url`, the custom browser binary path is:

```
/Applications/Brave Browser.app/Contents/MacOS/Brave Browser
```

This should already be configured in the Antigravity user settings. If the browser fails to open, remind the user to check that this path is set in their browser settings.
